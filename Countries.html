    This graph plots the cumulative cases, which means that it also includes closed cases (which either resulted in recovery, or death). Click the button below to only show active cases.
    This graph plots the calculated death rate each day.

    The death rate is calculated as follows:

    Death rate = #deathsday n / #active-casesday n

    Be aware that this calculation might not be very realistic, since the reported number of cases could be much lower than the actual cases.
    This graph plots the cumulative cases and the new cases since last week each day, on the x and y axis respectively. Be aware that the x axis does not indicate time and both axes are shown on logarithmic scale.

    On the x axis: #casesday n
    On the y axis: #new cases since last week

    This graph usually looks like a straight line, indicating that the number of cases is growing exponentially. When this straight line plummets down after a while, it means that the spread is slowing down.
    This graph plots the growth factor each day.

    A growth factor bigger than 1 implies that the number of active cases is growing, while a growth factor less than 1 implies the opposite.

    The growth factor is determined by the following formula:

    Growth Factor = #active-casesday n / #active-casesday n - 1
